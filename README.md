# Spotify Top Hits Playlist Generator

## Installation

Run the following to get the dependencies installed

```bash
  pip install -r requirements.txt
```

Add a .env file to root with the following defined from your Spotify App:
- CLIENT_ID = ''
- CLIENT_SECRET = ''
